# Atmos – Premium Atmospheric Weather Dashboard

Atmos is a high-performance, cinematic weather dashboard built with **React**, **TypeScript**, and **Tailwind CSS v4**. It features a custom-built atmospheric engine powered by **GSAP**, delivering a stunning visual experience with organic cloud animations, real-time lightning effects, and dynamic weather states.

![Atmos Preview](public/preview.png)

## 🌐 Live Demo
**Check out the live dashboard here:** [http://atmos-premium-weather.netlify.app/](http://atmos-premium-weather.netlify.app/)

## ✨ Key Features

- **🎬 Cinematic Atmospheric Engine**: Realistic, GSAP-animated organic SVG clouds that react to weather changes.
- **⚡ Real-time Dynamic Effects**: Integrated lightning strikes and rain animations synchronized with live weather data.
- **📊 High-Precision Data**: Powered by **Open-Meteo API**, providing accurate data for 81 major locations.
- **📉 Interactive Visuals**: Custom-designed wind speed bar charts and sunrise/sunset trajectories.
- **🎨 Modern Glassmorphism UI**: A sleek, dark-themed interface utilizing Tailwind CSS v4's latest utility engine.
- **📱 Responsive & Performance-First**: Optimized for all screen sizes with zero-dependency animations.

## 🛠️ Tech Stack

- **Framework**: React 19+ (Vite)
- **Styling**: Tailwind CSS v4
- **Animation**: GSAP (GreenSock Animation Platform)
- **Icons**: Custom minimalist "a" brand identity
- **API**: Open-Meteo (Free, no key required)

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/C3maLi/atmos-premium-weather.git
cd atmos-premium-weather
```

### 2. Install dependencies
```bash
pnpm install
# or
npm install
```

### 3. Start development server
```bash
pnpm dev
```

### 4. Build for production
```bash
pnpm build
```
